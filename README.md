# 💰 支票大写金额转换器

一个简洁、美观、实用的网页工具，用于将阿拉伯数字转换为繁体中文支票大写金额。

## ✨ 功能特点

- 🚀 **实时转换**：输入即转换，无需点击按钮
- 📋 **一键复制**：快速复制转换结果
- 💡 **智能提示**：提供常见示例，点击即可填充
- 📱 **响应式设计**：完美支持手机、平板和桌面端
- 🎨 **现代UI**：渐变色设计，视觉效果出色
- ⚡ **纯前端**：完全离线可用，无需网络连接
- 🔒 **隐私安全**：所有计算在本地完成，不上传任何数据

## 📖 使用说明

- **最大值**：999,999,999,999.99（千亿）
- **小数位**：最多两位（角、分）
- **不支持**：负数

## 📝 转换规则

### 数字映射

| 数字 | 繁体中文 | 数字 | 繁体中文 |
|------|---------|------|---------|
| 0 | 零 | 5 | 伍 |
| 1 | 壹 | 6 | 陸 |
| 2 | 貳 | 7 | 柒 |
| 3 | 叁 | 8 | 捌 |
| 4 | 肆 | 9 | 玖 |

### 单位规则

- **小单位**：拾、佰、仟
- **大单位**：萬、億
- **货币单位**：圓、角、分、整

### 零的处理

1. **中间有零需要读出**
   - `1001` → `壹仟零壹圓整`
   - `10086` → `壹萬零捌拾陸圓整`

2. **连续多个零只读一个**
   - `1000` → `壹仟圓整`（不是"壹仟零零零圓整"）

3. **小数部分的零**
   - 角为零但分不为零：`100.05` → `壹佰圓零伍分`
   - 角不为零分为零：`100.10` → `壹佰圓壹角`

4. **整数末尾加"整"**
   - `100` → `壹佰圓整`

## 🎯 核心功能实现

### 转换算法

核心转换逻辑位于 `converter.js`，主要包含：

1. **convertSection(num)**：处理万以下的数字转换
2. **convertInteger(num)**：处理整数部分，支持亿、万级别
3. **convertToChequeFormat(numStr)**：主转换函数，处理完整的转换逻辑

### 关键特性

- 智能零处理算法
- 支持大数字（千亿级别）
- 精确的小数处理（角、分）
- 完善的输入验证
- 错误提示机制
